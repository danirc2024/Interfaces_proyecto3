# Configuración

Esta carpeta contiene archivos de configuración para el proyecto.

## Archivos
- `mqtt_config.json` - Configuración del broker MQTT (host, puerto, credenciales)
- `dashboard_config.json` - Configuración general de dashboards
- `.env.example` - Ejemplo de variables de entorno (no incluir credenciales reales)

## Nota de Seguridad
⚠️ **NO** incluir credenciales reales en el repositorio. Usar variables de entorno o archivos .env (agregados al .gitignore).
